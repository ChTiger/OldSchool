In Teleconference, anything you type is broadcast to everyone on the same
channel with you. Also, use these special commands: -- SHORTHAND --
PAGE <who> ... "pages" the user with a message /P <who>
PAGE ON/OFF/OK allows/prevents/encourages others to page /P ON/OFF/OK
WHISPER TO <who> ... sends a private message to another user /<who> ...
CHAT <who> request or join in a "chat" with someone /C <who>
SCAN shows where everyone is in Teleconference /S
LIST lists all public channels you can join
EDIT edit your Teleconference profile /E
JOIN/JOIN <who> join a private, public, or forum channel /J <who>
INVITE <who> invites a user to your private channel /I <who>
UNINVITE <who> uninvites a user from your channel /U <who>
TOPIC ... list a topic for your private channel /T ...
